## Tstrul's DotFiles

### Usage:
1. Clone this repo:

```
git clone git@github.com:tstrul1/tstrul-dotfiles.git
cd tstrul-dotfiles
```


2. Run Backup & Install script:

```
./Install.sh
```

### Update dotfiles without apps:
Run symlinks script:

```
./createSymLinks.sh
```

## Take ad-hoc backup:

```
takeBackup.sh
```
